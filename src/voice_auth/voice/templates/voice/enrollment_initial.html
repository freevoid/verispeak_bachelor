  • Интернет-обозреватель, поддерживающий Java-апплеты (на официальном сайте Sun Microsystems можно проверить, поддерживаются ли апплеты в Вашем обозревателе);
  • Существующую учетную запись на сайте {% if user.is_authenticated %} (Вы зашли как {{ user.username }}); {% else %}{% if registration_opened %}(Регистрация);{% else %}(В данный момент свободная регистрация недоступна);{%endif%} {% endif%}
